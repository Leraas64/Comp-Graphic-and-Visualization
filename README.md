# Comp-Graphic-and-Visualization

#What new design skills has your work on the project helped you to craft?

While working on this project, I have learned how textures, shapes, and lighting can all affect one another. Each little aspect plays a part in the bigger picture.

#What design process did you follow for your project work?

I tried my best to imitate the sample photo as best as I could. I would first create the base shapes needed, then added some textures and finally tinkered with the lights to recreate the object to the best of my abilities.

#How could tactics from your design approach be applied in future work?

In most of my assignments, I find it easiest to create a good base before moving onto the next part. It's like trying to build a building, without a solid foundation you won't have a stable building.

#What new development strategies did you use while working on your 3D scene?

When tinkering with each setting, I would change one value to see how it affected the object and in what way. It was tough at first, but now that I have become more familiar with it, I can change more settings at once to get things the way I want faster.

#How did iteration factor into your development?

Iteration was the basis of my developmental process, it allowed me to learn how each setting changed everything.

#How has your approach to developing code evolved throughout the milestones, which led to the project's completion?

For the most part, my approach stayed about the same, but once I learned what each section of code was for, it became easier to find and change what I wanted.
